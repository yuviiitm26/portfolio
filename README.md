- **Interactive Elements**: Scroll animations, hover effects, and smooth scrolling.
- **Production Ready**: Semantic HTML, accessible contrast ratios, and optimized assets.

## How to Run

1.  **Simply Open the File**:
    Double-click `index.html` to open it in your default browser.

2.  **Using a Local Server (Recommended for best performance)**:
    If you have VS Code "Live Server" extension, right-click `index.html` and select "Open with Live Server".
    
    Or using python:
    ```bash
    python -m http.server
    ```
    Then open `http://localhost:8000`

## Customization

- **Colors**: Edit the `:root` variables in `style.css` to change the color scheme.
- **Content**: Update `index.html` with your real projects and experience.
- **Icons**: The site uses Devicon for tech stack logos. You can add more from [devicon.dev](https://devicon.dev/).
